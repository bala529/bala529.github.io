Botnet Surveillance System

In this day and age, peer to peer systems are becoming resilient and robust. To bring down an illegal peer to peer(p2p) network is next to impposible. With this networking technique, Botnets are spreading and infecting across the internet. This project was aimed at measuring and enumerating botnets on p2p network across the internet. This attempt would give new insights on the behaviour of several botnets around the internet and it can help us one step closer to make the internet safe today.

Semantic Data Collaboration

Data collaboration today needs to be future proof and research organisations need more than just bounded answers. Semantic data collaboration is a project to produce intelligent inferred results out of sparql queries. Inferred results of collaborated data gives us more insights on the future behaviour. Semantic web is supposed to be the so called "web 3.0" which links data from across the globe to produce a semantic rich search result. This project was done using the currently evolving Semantic web techniques like OWL and triple store to open doors for meaning full search results for a future oriented organisation. It was deployed completly with open source solutions and softwares.

IBM DB2-pureScale®

An internship at one of the world's oldest and largest IT pioneer. This internship involved learning the deployment phase of an enterprise grade database system. DB2 pureScale® feature was brought from already proven mainframe DB2 systems for ultra high-performance and high-availability networking and database. Purescale® is a state of the art feature to scale-out as many as 128 nodes on a single cluster using special hardware supported Remote Direct Memory Access protocol. The use of RDMA protocol reduces any kind of latency involved in the network stack, which significantly increased the performance of enterprise grade database system. This system was particularly implimented on IBM AIX platfrom and also on the Enterprise Linux platforms such as SUSE and RHEL.

Tools used

  • IBM AIX 7.1
  • RHEL,SUSE
  • IBM DB2 pureScale edition
  • Chef data center mangement
  • VMware Vsphere
  • LAMP stack for php website
  • IBM SAN
