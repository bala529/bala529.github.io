Botnet Surveillance System

In this day and age, peer to peer systems are becoming resilient and robust. To bring down an illegal peer to peer(p2p) network is next to impposible. With this networking technique, Botnets are spreading and infecting using this p2p network.

This project was aimed at measuring and enumerating botnets on p2p network. This attempt would give new insights on the behaviour of several botnets around the internet and it can help us one step closer to make the internet safe today.

Tools used

  • Docker
  • Ubuntu
  • Joyent Cloud
  • Java & Ant
  • Python
  • Ring Buffer
  • DNS load balancer(like DNS flux)
  • CPU affinity lock
  • Logstach-kibana-elasticsearch
  • nginx

Semantic Data Collaboration

Data collaboration today needs to be future proof and research organisations need more than just bounded answers. Semantic data collaboration is a project to produce intelligent inferred results out of sparql queries.

Inferred results of collaborated data gives us more insights on the future behaviour. Semantic web is supposed to be the so called "web 3.0" which links data from across the globe to produce a semantic rich search result.

This project was done using the currently evolving Semantic web techniques like OWL and triple store to open doors for meaning full search results for future oriented organisation.

Tools used

  • Docker
  • CentOS
  • RDF
  • Allegrograph
  • OWL java api
  • Pellet and Hermit reasoner
  • Karma
  • sparql
  • Gruff
