Botnet Surveillance system

In this day and age, peer to peer systems are becoming resilient and robust. To bring down an illegal peer to peer(p2p) network is next to impposible. With this networking technique, Botnets are spreading and infecting using this p2p network.

This project was aimed at measuring and enumerating botnets on p2p network. This attempt would give new insights on the behaviour of several botnets around the internet and it can help us one step closer to make the internet safe today.

Tools used

  • Docker
  • Ubuntu
  • Joyent Cloud
  • Java
  • Python
  • Ring Buffer
  • DNS load balancer(like DNS flux)
  • CPU affinity lock
  • Logstach-kibana-elasticsearch
  • nginx
